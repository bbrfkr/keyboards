# Quantum - 35%分割型キーボード
Quantumは以下の特徴を持つ、35%キーボードです。

- column-staggered
- 各指の長さに合わせた配列ズレにより、自然に手を置くだけで打鍵可能
- 文書やプログラム作成に必要十分な36キーのみを備え、手の移動を最小化
- Cherry MXとKailh low-profileスイッチに対応
- アンダーグローLEDおよび、OLEDに対応
