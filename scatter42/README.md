# Scatter42 - 40%分割型キーボード
Scatter42は以下の特徴を持つ、40%キーボードです。

- columnar-staggered
- 小指に無理をさせず、かつ親指を内側に入れる配置を避けるように意識したキー配列
- 制御キーの数も十分に取れる42キー
- Cherry MXスイッチに対応
